# Coding Challenge

## Getting Started

1) install dependencies
```
npm install
```
or 
```
yarn install
```

2) run webpack

```
npm run webpack
```
or 
```
yarn run webapck
```
or 
```
webpack --config ./front_end/config/webpack.dev.config.js
```

3) run docker

```
npm run docker
```
or 
```
yarn run docker
```
or 
```
sudo docker-compose -f ./docker/docker-compose.yml up
```
